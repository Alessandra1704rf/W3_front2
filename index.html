Explicación de conceptos css

1. Definición de CSS:

Lenguaje utilizado para describir cómo se presenta un HTML. Este proporciona reglas y estilos.

2. Explicar una regla de CSS:

Consiste de un selector y una declaración. En la declaración se encuentran propiedades (i.e color) y valores (i.e red). 

    h1 { //selector
        color: red; //declaración
    }

3. Explicar con ejemplos los 4 conceptos de CSS:

SELECTORES:

Dirige las REGLAS a elementos específicos.

    1) * {
    2) h1 {
    3) .elementoClass {
    4) #elementoID {

ESPECIFICIDAD:

Dicta qué estilo se aplicará en el caso de reglas CSS conflictivas que se dirigan al mismo elemento. La especificidad se calcula mediante la comparación de valores numéricos de los SELECTORES. El "mayor" es el estilo prominente.

    Etiqueta = 1
    Ejemplo -> h1 {

    Clase = 10
    Ejemplo -> .elementoClass {

    ID = 100
    Ejemplo -> #elementoID {

    Inline = 1000
    Ejemplo -> aplicado directamente a HTML utilizando el atributo "style".
    "style" permite definir 1 o más PROPIEDADES de CSS.

    !important = infinito
    Ejemplo -> .elementoClass {
        color: blue !important;
    }

CASCADA:

El último valor es el determinante mientras que la especificidad sea la misma.

    .elementoClass {
        color: blue;
    }

    .elementoID {
        color: green;
    }

    Resultado:
    green

HERENCIA:

Los estilos de "elementos padre" se propagan a elemenos descendientes.

        Ejemplo:

        HTML (sin < para que aparezca en web): 
        div
            a href = "#" Hola /a
        /div
        
        CSS:
        div{
            color: red;
        }
        a{
            color: inherit;
        }
